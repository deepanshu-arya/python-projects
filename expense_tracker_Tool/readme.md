# 💰 Expense Tracker Tool (Python)

A simple and efficient **Expense Tracker Tool** built using Python.  
This tool helps users **add, delete, view, and store expenses** in a clean and organized CSV file.

---

## 🚀 Features
- ➕ Add new expenses  
- ❌ Delete existing expenses  
- 📄 View all saved expenses  
- 💾 Auto-save data in a CSV file  
- 📊 Category & date-wise structure  
- 🧹 Clean and simple code (beginner-friendly)

---

## 🛠️ Technologies Used
- Python 3
- CSV module
- Datetime module
