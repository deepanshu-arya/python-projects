🔐 Python Password Generator

A simple, fast, and secure password generator built using Python.  
Generates strong passwords with uppercase letters, lowercase letters, numbers, and special characters.

---

## 🚀 Features
- Strong random password generation  
- Custom password length  
- Uses letters, digits & symbols  
- Lightweight and beginner friendly  
- Instant output  

---

## 🛠️ Technologies Used
- Python 3  
- Random module  
- String module
