# 📝 BlogSphere CMS API

**BlogSphere CMS API** is a simple and beginner-friendly FastAPI project for managing blog posts.  
It allows you to **Create**, **Read**, **Update**, and **Delete** (CRUD) blog entries using a RESTful API.

---

## 🚀 Features
- Add new blog posts  
- View all blog posts  
- View a single blog post by ID  
- Update existing posts  
- Delete posts  
- Stores data in SQLite  
- Fast, clean, and easy to understand  

---

## 🧠 Tech Stack
- **FastAPI** — Web framework  
- **SQLAlchemy** — ORM  
- **SQLite** — Lightweight database  
- **Uvicorn** — ASGI server
