# 🚚 OrderTrack Delivery System (FastAPI Project)

A simple and beginner-friendly **FastAPI** project to manage and track delivery orders.

---

## 🌟 Features
- Add new orders  
- View all orders  
- Get order details by ID  
- Update delivery status  
- Delete orders  
- Clean, simple, and fully error-free code  

---

## 🏗️ Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Pydantic**
- **Uvicorn**
