# 🤖 ChatWise AI Bot (FastAPI Project)

A beginner-friendly **FastAPI chatbot** project that replies to basic user messages using simple Python logic.  
No external AI APIs — just clean, understandable code.

---

## 🌟 Features
✅ Simple and interactive chatbot  
✅ Responds to greetings, date, time, and casual messages  
✅ Clean FastAPI-based REST API  
✅ Easy to run and extend with your own logic  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Pydantic**
- **Uvicorn**
