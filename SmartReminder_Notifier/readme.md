# 🕒 SmartReminder Notifier (FastAPI Project)

A simple **FastAPI-based reminder system** where you can schedule reminders that automatically notify you in the console when it's time.  

Perfect for beginners learning **FastAPI** and background tasks.

---

## 🌟 Features
✅ Create new reminders  
✅ View all scheduled reminders  
✅ Auto-delete reminders after they trigger  
✅ Background notification using Python threads  
✅ Simple, clean, and fully error-free code  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Uvicorn**
- **Pydantic**
