# 🏪 AutoStock Manager (FastAPI Project)

A simple and easy-to-understand **FastAPI project** for managing product inventory and automating stock updates.

---

## 🌟 Features
✅ Add new products  
✅ View all products  
✅ Get details by product ID  
✅ Update stock manually  
✅ Automatically reduce stock after sales  
✅ Delete products  
✅ Simple, clean, and error-free code  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Pydantic**
- **Uvicorn**
