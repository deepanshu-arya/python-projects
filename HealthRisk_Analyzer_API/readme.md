# 🧠 HealthRisk Analyzer (Predictive API Model)

A simple **FastAPI + Machine Learning** project that predicts whether a person has a **high** or **low health risk** (like diabetes) based on input parameters such as age, BMI, glucose, blood pressure, and insulin levels.

---

## 🚀 Features
- FastAPI backend for real-time predictions
- Logistic Regression model for health risk prediction
- Automatic model training if no model file exists
- Clean and easy-to-understand code
