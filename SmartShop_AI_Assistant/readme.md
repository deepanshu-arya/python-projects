# 🛍️ SmartShop AI Assistant (Shop Management Backend)

A simple **FastAPI-based backend system** for managing products, customers, and sales in small shops.  
This project helps shopkeepers handle **product inventory, sales tracking, and customer management** — all through simple API endpoints.

---

## 🚀 Features
- Add, update, delete, and list products  
- Add and list customers  
- Record transactions and auto-update stock  
- Get total sales summary  
- Simple, clean, and understandable code
