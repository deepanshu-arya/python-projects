# 💰 PriceInsight Analyzer (FastAPI Project)

A beginner-friendly **FastAPI project** that analyzes product prices and provides insights such as:
- Average price  
- Highest and lowest prices  
- Cheapest and costliest products  

---

## 🌟 Features
✅ Analyze a list of product prices  
✅ Get average, highest, and lowest price instantly  
✅ Simple and understandable FastAPI code  
✅ Perfect project for GitHub portfolio  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Uvicorn**
- **Pydantic**
