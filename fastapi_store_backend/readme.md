# 🏪 FastAPI Store Backend

**FastAPI Store Backend** is a simple CRUD (Create, Read, Update, Delete) API for managing store products.  
It uses **FastAPI** with **SQLite** as the database and **SQLAlchemy** as ORM.

---

## 🚀 Features
- Add new products
- View all products
- View a single product
- Update existing product details
- Delete products
- Uses SQLite (local database)
- Simple, clean, and beginner-friendly

---

## 🛠️ Requirements
- Python 3.8 or above  
- FastAPI  
- SQLAlchemy  
- Uvicorn (for running the server)
