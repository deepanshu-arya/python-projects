# 🔐 SecureAuth Login System (FastAPI Project)

A simple and beginner-friendly **FastAPI-based login system** where users can **register** and **log in** using basic authentication.  
This version uses an **in-memory dictionary (no database)** — perfect for beginners learning FastAPI authentication basics.

---

## 🌟 Features
✅ Register new users  
✅ Login with username and password  
✅ Prevent duplicate usernames  
✅ Simple, clear, and error-free code  
✅ Great for understanding FastAPI fundamentals  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Pydantic**
- **Uvicorn**
