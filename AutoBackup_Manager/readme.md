# 💾 AutoBackup Manager (FastAPI Project)

A simple and beginner-friendly **FastAPI project** that automatically backs up files from a source folder to a backup folder at regular intervals.

---

## 🌟 Features
✅ Start automatic backups  
✅ Backup files from source → destination folder  
✅ View running status  
✅ Stop backup anytime  
✅ Error-free and easy to understand  

---

## 🧠 Tech Stack
- **Python 3.9+**
- **FastAPI**
- **Uvicorn**
- **Pydantic**
- **Shutil & Threading (Python Standard Library)**
